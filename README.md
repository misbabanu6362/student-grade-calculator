# student-grade-calculator
Student Grade Calculator

A beginner Python project that calculates a student's total marks, average marks, and grade based on marks entered by the user.

Features

- Takes student name as input
- Accepts marks for three subjects
- Calculates total marks
- Calculates average marks
- Assigns grade based on average
- Displays student details and results

Concepts Used

- Variables
- Input and Output
- Data Types
- Type Conversion
- Operators
- Conditional Statements (if, elif, else)

Grade Criteria

Average Marks| Grade
90 and above| A
75 - 89| B
60 - 74| C
Below 60| D

Sample Output

Enter Student Name: Aftab
Enter Marks in Subject 1: 80
Enter Marks in Subject 2: 90
Enter Marks in Subject 3: 85

Total Marks: 255
Average Marks: 85.0
Grade: B
